# Ruboty::Ghibli

A [Ruboty](https://github.com/r7kamura/ruboty) plugin.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-ghibli'
```

And then execute:

    $ bundle install

## Usage

```
ruboty ghibli
```
